# Utils (Legacy)

Package utilities now live in `polydiff/utils/` inside the Python package.
This folder is reserved for non-package helpers if needed.

Current variable-size polygon support is implemented in package modules such as:

- `polydiff/data/polygon_dataset.py`
- `polydiff/models/diffusion.py`
- `polydiff/training/train.py`
- `polydiff/sampling/sample.py`
